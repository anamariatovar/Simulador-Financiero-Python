# Simulador-Financiero-Python

Este proyecto consiste en el desarrollo de un simulador financiero en Python, basado en un escenario hipotético en el que 100 familias, afectadas económicamente, acceden a un préstamo bajo las mismas condiciones iniciales: un mismo capital y una tasa de interés fija.

La única variable que cambia es el período de pago, permitiendo analizar cómo la variación en el tiempo impacta el valor total a pagar debido a la acumulación de intereses.

A partir de esta situación, se implementa un modelo matemático fundamentado en el cálculo integral, con el objetivo de estudiar el comportamiento del interés a lo largo del tiempo y evidenciar las diferencias entre distintos plazos de financiamiento.

Como resultado, se propone una herramienta denominada “semáforo financiero”, la cual traduce los resultados matemáticos en una representación visual sencilla (niveles de riesgo), facilitando la comprensión del impacto financiero y apoyando la toma de decisiones informadas.

El sistema está diseñado de forma flexible, permitiendo modificar variables como capital, tasa de interés y tiempo, lo que lo hace aplicable a diversos contextos, como análisis de créditos, proyección de inversiones y planificación financiera.
